Tabletop Helper

This application is an companion application for board games. It's designed for mobile use, where all data and objects should be cached on your device, so it should work without using the Internet. If you want to report any issue or request some functionality or game to be added, please do so via the Issue tab on my GitHub.

Link to the project on GitHub is: https://github.com/Stinolez/Tabletop-Helper
